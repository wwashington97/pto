A simple PTO calc that uses how many years you worked to get a correct PTO rate. Then uses how many weeks you have worked so far this year to calculate total PTO to date. Then deducts any PTO you have taken to date.

  1. START
  2. Declarations
    1. wksWrkd
    2. yrsWrkd
    3. hrsPerCk
    4. totHrsToDate
    5. totPTO
    6. hrsLeft
  3. Display "How many years have you worked?"
  4. INPUT yrsWrkd
  5. If yrsWrkd is greater than 0 but less than 5 , hrsPerCk = 5
  6. If yrsWrkd is 5 or greater, hrsPerCk = 6
  7. If yrsWrkd is 10 or greater, hrsPerCk = 7
  8. Set totPTO = hrsPerCk * 26
  9. Display "How many weeks have you worked so far this year?"
  10. INPUT wksWrkd
  11. Set totHrsToDate = hrsPerCk * wksWrkd
  12. Set hrsLeft = totPTO - totHrsToDate
